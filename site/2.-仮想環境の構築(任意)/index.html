    2. 仮想環境の構築(任意)

    2.1. Windows 10のisoファイルをダウンロードする

    Windows 10 isoファイルのダウンロード

    1. ツールを今すぐダウンロードをクリックし、MediaCreationTool21H2.exeをダウンロードして開く。

    2. 同意するをクリック。

    3. 別のPCのインストールメディアを作成するを選択し、次へをクリック。

    4. 以下の設定を行い、次へをクリック。

      • 言語:日本語
      • エディション:Windows 10

      • アーキテクチャ:64ビット (x64)

    5. ISOファイルを選択し、次へをクリック。

    6. 保存をクリック ここではドキュメントフォルダー内に保存しますが、別フォルダーに保存しても問題ありません。

    7. 完了をクリック

    以上でWindows 10のisoファイルのダウンロードが完了しました。

    2.2. VMwareのインストール

    VMware Workstationをダウンロード

    1. GO TO DOWNLOADSをクリック。

    1. 上のVMware Workstation 16.2.3 Player for Windows 64-bit Operating Systemsの横にあるDOWNLOAD NOWをクリックし、ダウンロードして開く。

    1. 次へをクリック。

    2. 使用許諾契約書に同意しますをチェックし、次へをクリック。

    3. 次へをクリック。

    4. 次へをクリック。

    5. 次へをクリック。

    6. インストールをクリック。

    7. 完了をクリック。

    8. 再起動しなくても問題ないのでいいえをクリック。

      以上でVMwareのインストールが完了しました。

    2.3. VMware上でWindows 10を動かす

    1. デスクトップに追加されたVMware Workstation 16 Playerをダブルクリック。

    2. 新規仮想マシンの作成をクリック。

    3. Windows 10 x64が検出されましたと表示された場合、次へをクリック。

    4. 何も表示されない場合、参照をクリック。

    5. 2.1. Windows 10のisoファイルをダウンロードするでダウンロードしたWindows.isoを選択し、開くをクリック。

    6. 次へをクリック。

    7. 次へをクリック。

    8. デフォルトでは60GBに設定されていますが、40GB程でも問題なく動作します。ここでは60GBに設定し、次へをクリック。

    9. ハードウェアをカスタマイズをクリック。

    10. 自分のPCスペックを見るために、Win + Rキーを同時押しし、dxdiagと入力。

    11. メモリを設定します。

      DirectX診断ツールのメモリの欄が、現在お使いのPCに搭載されているメモリの量です。

      搭載されているメモリの半分を設定しましょう。ここでは8192MBと設定します。

    12. プロセッサ数を設定します。

      DirectX診断ツールのプロセッサの欄が、現在お使いのPCに搭載されているプロセッサです。16 CPUs（搭載されているプロセッサによって異なります）と書かれているのがプロセッサ数です。

      搭載されているプロセッサ数の半分を設定しましょう。ここでは8と設定します。

    13. 設定し終えたら閉じるをクリック。

    14. 完了をクリック。

    15. OKをクリック。

    16. VMwareの画面をクリックし、すぐにどのキーでもいいので押してください。VMwareの画面から戻るには、CTRL + ALTキーを同時押し。

    17. キーを押すのが間に合わなかった場合、Boot normallyを選択し、エンターキーを押す。番号16に戻ります。

    18. 次へをクリック。

    19. 今すぐインストールをクリック。

    20. プロダクトキーがありませんをクリック。

    21. Windows 10 Proを選択し、次へをクリック。

    22. 同意しますにチェックを入れ、次へをクリック。

    23. 下のカスタム:Windows のみをインストールするをクリック。

    24. 次へをクリック。インストールが開始されるので、のんびり待ちましょう。

    25. はいをクリック。

    26. はいをクリック。

    27. スキップをクリック。

    28. 個人用に設定を選択し、次へをクリック。

    29. オフラインアカウントをクリック。

    30. 制限付きエクスペリエンスをクリック。

    31. ユーザー名を半角英数字（例:hogemaru, aiueoなど）で入力し、次へをクリック。

    32. パスワードを入力し、次へをクリック。

    33. 番号32で入力したパスワードを再度入力し、次へをクリック。

    34. セキュリティの質問を適当に設定。

    35. お好みでオフにし、同意をクリック。

    36. スキップをクリック。

    37. 今は実行しないをクリック。

      以上でWindows 10のインストールが完了しました。

    2. 右下の通知をクリック。

    3. setup64.exeの実行をクリック。

    4. 次へをクリック。

    5. 次へをクリック。

    6. インストールをクリック。

    7. 完了をクリック。

    8. いいえを押しても問題ないです。

      以上でVMware Toolsのインストールが完了しました。
