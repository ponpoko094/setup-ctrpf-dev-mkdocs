    3. CTRPFの開発環境を構築する

    3.1. 前準備

    3.1.1. Nanquitas's Playgroundに参加する

    ※必要なファイルをダウンロードするために、DiscordサーバーのNanquitas's Playgroundに参加してください。

    1. ユーザー名を入力してはいをクリック。

    2. 私は人間ですにチェックを入れる。

    3. なんとかして解く

    4. 生年月日を入力して完了をクリック。

    5. バツマークをクリック

      以上でサーバーに参加できました。

    3.1.2. 7-zipのインストール

    必要なファイルがtar.bz2という拡張子で配布されていて、デフォルトのままだと展開できないため、7-zipをインストールしてください。

    1. 7-zipにアクセス。

    2. 一番上のダウンロードをクリックし、開く。

    3. Installをクリック。

    4. Closeをクリック。

      以上で7-zipのインストールが完了しました。

    3.1.3. エクスプローラーの設定

    1. エクスプローラーを開く。

    2. 表示タブをクリック。

    3. 以下の設定にする。

    • 項目チェックボックスにチェックを入れない
    • ファイル名拡張子にチェックを入れる
    • 隠しファイルにチェックを入れる

      以上でエクスプローラーの設定が完了しました。

    3.2. devkitProのインストール

    1. Microsoft Edgeをダブルクリック。

    2. devkitproと検索。

    3. Releases・devkitPro/installer・GitHubをクリック。

    4. devkitProUpdater-3.0.3.exeをクリックし、ダウンロード。

    5. ダウンロードが完了したら開く。

    6. Nextをクリック。

    7. Nextをクリック。

    8. Nextをクリック。

    9. Nextをクリック。

    10. Nextをクリック。

    11. Installをクリック。

    12. Nextをクリック。

    13. Finishをクリック。

      以上でdevkitProのインストールが完了しました。

    3.3. libctrpf-0.7.2のインストール

    1. info_releasesチャンネルをクリック。

    2. libctrpf-0.7.2.eee6105a.tar.bz2をクリックし、ダウンロード。

    3. Win + Eキーを同時押しし、エクスプローラーを開く。

    4. ダウンロードをクリック。

    5. **libctrpf-0.7.2.eee6105a.tar.bz2を右クリックし、7-zip→展開…をクリック。

    6. OKをクリック。

    7. libctrpf-0.7.2.eee6105a.tarをダブルクリック。

    8. libctrpf-0.7.2.eee6105a.tarを右クリックし、7-zip→展開…をクリック。

    9. OKをクリック。

    10. libctrpf-0.7.2.eee6105aをダブルクリック。

    11. 2つのフォルダーを選択して、CTRL + Cキーでコピー。

    12. 左の欄のPCをクリック。

    13. ローカルディスクをダブルクリック。

    14. devkitProをダブルクリック。

    15. CTRL + SHIFT + Nキーで新規フォルダーを作成。名前はlibctrpfにする。

    16. libctrpfをダブルクリック。

    17. CTRL + Vキーで貼り付ける。

      以上でlibctrpfのインストールが完了しました。

    3.4. 3gxtool.exeのインストール

    1. 3gxtool.exeをクリックし、ダウンロード。

    2. 3gxtool.exeを選択し、CTRL + Cでコピー。

    3. PC→ローカルディスク→devkitPro→tools→binをダブルクリック。

    4. CTRL + Vキーで貼り付ける。

      以上で3gxtool.exeのインストールが完了しました。

    3.5. CTRPFソースのダウンロード

    1. CTRPluginFramework-BlankTemplateをクリック。

    2. 緑色のCodeをクリック、Download ZIPをクリックし、ダウンロード。

    3. ダウンロードフォルダーを開いて、CTRPluginFramework-BlankTemplate-master.zipを右クリックし、7-zip→展開…で展開をする。

    4. CTRPluginFramework-BlankTemplate-masterの中にもう1つ同名のフォルダーがあるので、それをコピーする。

    5. PC→ローカルディスク→ユーザー→ユーザー名→を開き、Programという名前でフォルダーを作る。

    6. Programの中にCTRPFという名前でフォルダーを作る。

    7. CTRPFの中にCTRPluginFramework-BlankTemplate-masterをコピーする。

      以上でCTRPFのソースのダウンロードが完了しました。
